 There are thousands, if not millions, of application programs in everyday use that automatically generate texts; but probably fewer than ten of these programs use the linguistic and knowledge-based techniques that have been studied by the Natural-Language Generation (NLG) community.  The other 99.9% of systems use programs that simply manipulate character strings, in a way that uses little, if any, linguistic knowledge.
